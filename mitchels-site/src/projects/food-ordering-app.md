### Stack

Front End: JQuery\
Back End: Express, NodeJS
Database: postgreSQL\
CSS Preprocessor: Sass\
APIs: Twilio

### Description

A food ordering app created as a midterm project for the Lighthouse Labs Web Development Bootcamp. The app utilizes JQuery and allows customers to place an order for pickup at a single restaurant. Both a client and restaurant side were created and both sides receive a text notification when an order is place.
